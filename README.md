# PR. 2 Analyzer

## Project Overview

Analyzer is an Advanced Microsoft Excel project developed to analyze customer sales data using formulas, conditional formatting, Pivot Tables, charts, Data Analysis ToolPak, and dashboard techniques. The objective of this project is to transform raw sales data into meaningful business insights through visualization and analytical methods.

---

## Project Objectives

- Analyze customer sales performance.
- Identify high-value customers.
- Measure monthly sales growth.
- Perform statistical analysis using Excel Data Analysis ToolPak.
- Analyze the relationship between Sales and Profit using Linear Regression.
- Build an interactive dashboard with KPI cards and charts.
- Present business insights in a professional format.

---

The dataset contains the following fields:
